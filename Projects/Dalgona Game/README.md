# 🦑 Dalgona Game Challenge (Linux Shell Script Edition)

Inspired by the iconic challenge from *Squid Game*, this Shell Script-based **Dalgona Game Challenge** brings the tension, skill, and thrill of cutting out delicate shapes — all within your Linux terminal!

## 🎮 Features

- **🧍 Single Player Mode** – Play solo and test your nerve.
- **🤖 Player vs Computer Mode** – Compete against a bot and see who survives the sugar-cracking pressure!
- **⏱️ Dynamic Timer** – Time limits based on difficulty:
  - 🟢 Easy
  - 🟡 Medium
  - 🔴 Hard
- **❤️ 3 Chances** – You have three tries to complete the challenge.
- **🔢 Scoring System** – Earn points for successful cuts. Lose points for mistakes or breaking the shape.
- **🎲 Random ASCII Shapes** – Circle, Star, Umbrella, Triangle, and more — each presented in ASCII art.

## 🛠️ Requirements

- Linux OS (Tested on Ubuntu/Debian)
- `bash` shell
- `tput` (for terminal control)

## 🚀 How to Run

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/MadG0blin359/SH-Scripts

2. **Navigate to the Project Directory:**

   ```bash
   cd SH-Scripts/Projects/'Dalgona Game'

3. **Make the Script Executable:**

   ```bash
   chmod +x Dalgona.sh

4. **Run the Executable:**

   ```bash
   ./Dalgona.sh
